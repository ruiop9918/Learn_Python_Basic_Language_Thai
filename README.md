# โครงงานพัฒนาสื่อเพื่อการศึกษา
### *เรื่อง สื่อการสอนภาษาโปรแกรม Python*

---

> ### ความสำคัญของโครงงาน
ในปัจจุบัน ภาษาโปรแกรม **Python** ได้รับความนิยมอย่างแพร่หลาย ทั้งในด้านการศึกษา การพัฒนาเว็บไซต์ ปัญญาประดิษฐ์ (AI) และการวิเคราะห์ข้อมูล เนื่องจากมีไวยากรณ์ที่เข้าใจง่าย เเละเหมาะสำหรับผู้เริ่มต้นเรียนรู้การเขียนโปรแกรม  
ดังนั้นคณะผู้จัดทำจึงมีแนวคิดในการพัฒนา **สื่อการสอนภาษาโปรแกรม Python** เพื่อให้ผู้เรียนสามารถศึกษาและทำความเข้าใจพื้นฐานการเขียนโปรแกรมได้ด้วยตนเอง **ผ่านสื่อเว็บไซต์**

---

> ###  วัตถุประสงค์ของโครงงาน
1. เพื่อพัฒนาเว็บไซต์หรือสื่อการสอนที่ช่วยให้ผู้เรียนเข้าใจหลักการพื้นฐานของภาษา Python  
2. เพื่อส่งเสริมการเรียนรู้ด้วยตนเองผ่านสื่ออินเทอร์แอคทีฟ  
3. เพื่อใช้เทคโนโลยีสมัยใหม่ในการจัดการเรียนการสอนให้มีประสิทธิภาพมากขึ้น  
4. เพื่อกระตุ้นความสนใจของผู้เรียนในด้านวิทยาการคอมพิวเตอร์และการเขียนโปรแกรม
5. เพื่อสร้างพื้นฐานให้ผู้เรียนรู้ เเละสามารถต่อยาดความรู้ที่สูงขึ้นเองได้ครับ
---

> ###  ขอบเขตของโครงงาน
- เนื้อหาครอบคลุมพื้นฐานของภาษา Python เช่น  
  - การแสดงผลและรับข้อมูล  
  - ตัวแปรและชนิดข้อมูล  
  - การใช้เงื่อนไขและลูป  
  - ฟังก์ชันเบื้องต้น  
  - พื้นฐาน OOP   
-   ผู้ใช้สามารถเรียนรู้เนื้อหาบนสื่อเว็บไซต์ของเราได้ฟรี
